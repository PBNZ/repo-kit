# Code of Conduct

## Our commitment

We want this project to be a welcoming, respectful, and productive space for everyone — regardless of background, identity, or experience level.

## Expected behaviour

Participants in this project are expected to:

- Be respectful, patient, and generous in interpretation.
- Offer and accept constructive feedback gracefully.
- Acknowledge mistakes, apologise when appropriate, and move forward.
- Focus on what is best for the project and its users.
- Use clear, professional language in issues, pull requests, commit messages, and discussions.

## Unacceptable behaviour

The following will not be tolerated in any project space:

- Personal attacks, insults, or demeaning comments directed at other participants.
- Discriminatory or exclusionary language or conduct of any kind.
- Unwanted personal attention or contact.
- Publishing another person's private information without their explicit consent.
- Sustained disruption of discussions, issues, or reviews.
- Any other conduct that a reasonable person would find inappropriate in a professional setting.

## Scope

This Code of Conduct applies to all project spaces — including issues, pull requests, commit messages, code review comments, and any other forum connected to this repository — and to any public spaces where an individual is representing the project.

## Reporting

If you witness or experience behaviour that violates this Code of Conduct, please report it privately to the maintainers using GitHub's **Report abuse** flow on the relevant user, issue, or comment, or by opening a [private security advisory](https://github.com/PBNZ/repo-kit/security/advisories/new) (which is a private channel and an appropriate route for sensitive reports of this kind).

All reports will be reviewed and handled confidentially. The maintainers are committed to respecting the privacy and safety of anyone who reports an incident.

## Enforcement

Maintainers are responsible for clarifying and enforcing this Code of Conduct. They may take any action they deem appropriate, including:

- Removing, editing, or rejecting contributions that do not align with this Code of Conduct.
- Issuing warnings.
- Temporarily or permanently banning contributors for behaviour they judge to be inappropriate, threatening, or harmful.

## Attribution

This Code of Conduct is inspired by the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/) and similar community codes. It has been written in the project's own words to keep reporting channels aligned with how this project actually operates.
